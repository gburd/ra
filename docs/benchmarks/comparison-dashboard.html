Generate Your Own Results

To create a dashboard with real benchmark results from your system:

# Run all benchmarks and generate HTML dashboard
ra-cli benchmark --all --format html --output comparison.html

# Open the dashboard in your browser
open comparison.html

# Or run specific database/workload combinations
ra-cli benchmark --database postgresql --workload hybrid-search

# Run the automated benchmark suite
./scripts/run-all-benchmarks.sh

Command Options

  • --all - Run all database and workload combinations
  • --database - Specify database: postgresql, mysql, sqlite, duckdb
  • --workload - Specify workload: hybrid-search, vector-search, fts, joins, aggregates, analytics
  • --format - Output format: html, json, markdown
  • --output - Output file path

Example Output

The generated dashboard will include:

  • Interactive charts showing execution time comparisons
  • Speedup distribution across queries
  • Rows scanned comparison (native vs Ra)
  • Complexity vs speedup scatter plots
  • Side-by-side query plan comparison
  • Filterable results table
  • Export functionality for further analysis
