## How to Acknowledge

You acknowledge this agreement by one of the following methods:

### Option A: PR Comment

Add this comment to your first pull request:

> I have read and agree to the
> [Contributor License Agreement](CONTRIBUTOR_AGREEMENT.md).

### Option B: Signed Commits

Sign your commits using the Developer Certificate of Origin (DCO):

```bash
git commit -s -m "Your commit message"
```

This adds a `Signed-off-by` line to your commit, indicating you
certify the [Developer Certificate of Origin](https://developercertificate.org/):

```
Signed-off-by: Your Name <your.email@example.com>
```

### Option C: PR Checkbox

Check the CLA acknowledgment box in the pull request template when
submitting your PR.

Any of these methods constitutes acceptance of this agreement. You only
need to acknowledge the agreement once; it applies to all future
contributions.
